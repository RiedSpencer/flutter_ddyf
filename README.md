## 项目说明

模仿小程序 “点点缘分” 做的一个基础flutter版的项目，不完善，但是基础布局和部分交互已实现，主要可以参考的功能点如下：

- 导航栏
- tab切换
- 照片上传
- 网络请求
- 父子组件状态变更
- slider滑动
- 路由跳转
- 基本交互

可以直接 build 生成apk，在手机安装之后查看效果     
或者直接 down 在 build\app\outputs\apk\release\app-release.apk 安装查看效果
